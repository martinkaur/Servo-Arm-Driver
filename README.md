# Servo driver firmware for Servo Arm Controller project
To be opened for editing in STM32CubeIDE.

## Uploading firmware and debugging
STM32CubeProgrammer alongside STlink and JTAG/SW pins are used to upload firmware or debug.
Firmware will be executed as specified in the programmer or after hardware reset (button on controller PCB or power cycle).
Pins on the controller PCB are labelled accordingly.
